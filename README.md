# Bingo Fit

A gamified fitness app, using the style of a popular traditional game in western culture - Bingo!

## Getting started

### Installing dependencies
```
cd skeleton
npm i

```

### Seed the database
```
npx prisma db seed

```

### Edit the database
```
npx prisma studio
```
